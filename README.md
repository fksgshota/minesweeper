# Minesweeper

モダンなデザインとサウンドエフェクトを備えた、フル機能のマインスイーパーゲームです。

## ✨ 特徴

- 🎮 **4つの難易度レベル**
  - 初級: 9×9 (10個の地雷)
  - 中級: 16×16 (40個の地雷)
  - 上級: 30×30 (120個の地雷)
  - マニア: 68×68 (777個の地雷)

- 🎨 **モダンなUI/UX**
  - ダークモードのグラスモーフィズムデザイン
  - スムーズなアニメーションとグラデーション
  - レスポンシブ対応 (PC・タブレット・スマホ)

- 🎵 **サウンドシステム**
  - Web Audio APIによる音声生成
  - BGM (コード進行: Am-F-C-G)
  - 効果音 (クリック、開示、旗、爆発、クリア)
  - オン/オフ切り替え可能

- 🏆 **記録管理**
  - 難易度ごとのベストタイム保存
  - ランキング表示 (金・銀・銅メダル)
  - LocalStorageで永続化

- 🎯 **ゲーム機能**
  - 左クリック: セル開示
  - 右クリック: 旗の設置/解除
  - 旗モード切り替え
  - 一時停止/再開
  - カウントアップタイマー
  - カスタムモーダルダイアログ

## 🚀 使い方

1. ブラウザで `index.html` を開く
2. 難易度を選択
3. セルをクリックしてゲーム開始
4. すべての地雷を回避してクリアを目指す

## 📱 対応環境

- モダンブラウザ (Chrome, Firefox, Safari, Edge)
- デスクトップ・タブレット・モバイル
- Web Audio API対応ブラウザ

## 🎮 操作方法

### マウス操作

- **左クリック**: セルを開く
- **右クリック**: 旗を立てる/外す

### ボタン操作

- **難易度変更**: 難易度を順番に切り替え
- **一時停止**: ゲームを一時停止/再開
- **旗切り替え**: 左クリックで旗モードに切り替え
- **リセット**: 現在の難易度でゲームをリセット
- **🔊/🔇**: サウンドのオン/オフ

## 🏅 記録システム

- 各難易度のクリアタイムを自動記録
- 新記録達成時に通知
- モーダルでベスト3を表示
- ランク別バッジ表示 (🥇🥈🥉)

## 💡 アルゴリズム

- **地雷配置**: Fisher-Yatesシャッフル
- **セル開示**: 再帰的なフラッドフィルアルゴリズム
- **音声生成**: オシレーター (Sine, Square, Triangle, Sawtooth)
